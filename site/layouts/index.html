Selamat Datang di Website KP2KP Tomohon

KP2KP Tomohon akan memberikan pelayanan terbaik kepada Wajib Pajak. Website ini dibuat sebagai salah satu bentuk pelayanan kepada Wajib Pajak. Dengan menggunakan website KP2KP Tomohon ini, diharapkan Wajib Pajak bisa lebih mudah untuk menghubungi KP2KP Tomohon.

Dengan website ini, Wajib Pajak dapat langsung menghubungi petugas melalui saluran chat

#SudahPunyaTapiBelum

Sudah punya NPWP tapi belum lapor pajak? Kini pelaporan bisa dilakukan melalui situs pajak.go.id. Bagi yang masih kurang mengerti bagaimana melakukan pelaporan secara e-filing bisa kami pandu melalui website ini.
